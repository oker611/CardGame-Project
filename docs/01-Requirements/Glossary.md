## 术语表 (Glossary)

| 术语 | 英文 | 定义 |
| ---- | ---- | ---- |
| 计分规则 | Scoring | 游戏结束后系统根据玩家剩余手牌数量、牌型计算分输并用于判定胜负即排名。 |
| 手牌 | Hand Cards | 玩家当前持有的所有扑克牌，游戏开始时每人 13 张，出牌后相应减少。 |
| 回合 | Turn | 从当前玩家开始操作（出牌或过牌）到系统切换到下一位玩家之间的过程。一局游戏由多个回合组成。 |
| 出牌 | Play Cards | 玩家在回合内选择若干张手牌并打出，需符合牌型规则且能压过上家（若首轮则无压牌要求）。 |
| 过牌 | Pass | 玩家在回合内选择不出牌，放弃本轮出牌权，将回合切换给下一位玩家。 |
| 牌型 | Card Pattern | 单张、对子、顺子、同花、葫芦、炸弹等合法组合方式。不同牌型有不同的大小比较规则。 |
| 校验 | Validation | 系统检查玩家所出牌是否合法（牌型正确、大小满足压制条件），是出牌流程的必要环节。 |
| 结算 | Settlement | 一局游戏结束后，系统根据玩家出完手牌的先后顺序判定排名，并展示胜负结果。 |
| 房间 | Room | 房主通过蓝牙创建的虚拟空间，其他玩家可加入，房间内所有玩家共享同一局游戏状态。 |
| 房主 | Host | 创建房间的玩家，拥有设置游戏规则、开始游戏等特殊权限。 |
| 发牌 | Deal Cards | 游戏开始时，系统将 52 张牌随机平均分给所有玩家，并标记持有方块 3 的玩家为首轮出牌者。 |
